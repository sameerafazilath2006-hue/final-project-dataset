Social Media Usage & Mental Health Analysis
📌 Project Overview

This project analyzes the relationship between social media usage and mental health factors among students/young individuals.
The dataset contains information about screen time, sleep patterns, stress, anxiety, addiction levels, academic performance, and depression indicators.

The main goal of this project is to identify how excessive social media usage may affect mental and physical well-being.

📂 Dataset Information
Dataset Name: Social Media & Mental Health Dataset
Rows: 1200
Columns: 13
Features Included
Column Name	Description
age	Age of the individual
gender	Gender of the individual
daily_social_media_hours	Daily time spent on social media
platform_usage	Social media platform used
sleep_hours	Average sleep hours
screen_time_before_sleep	Screen time before sleeping
academic_performance	Academic performance score
physical_activity	Daily physical activity level
social_interaction_level	Social interaction category
stress_level	Stress level score
anxiety_level	Anxiety level score
addiction_level	Social media addiction level
depression_label	Depression indicator (0/1)
🎯 Problem Statement

Excessive use of social media can influence sleep quality, stress, anxiety, academic performance, and overall mental health.
This project aims to analyze these relationships using data analytics techniques.
